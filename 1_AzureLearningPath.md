
# Azure Learning

## Azure Learning Path from Microsoft
1. Navigate to the link https://docs.microsoft.com/en-us/learn/browse/?products=azure%2Csql-server
2. Select your products from left navigation
3. <img src="../main/MediaFiles/Azure/AzProductFilter.png" width="40%">
4. Select a role that you are looking for
5. <img src="../main/MediaFiles/Azure/AzRoleFilter.png" width="40%">
6. Select Level
7. <img src="../main/MediaFiles/Azure/AzLevelsFilter.png" width="40%">
8. Result will be filtered based on your selection
9. <img src="../main/MediaFiles/Azure/AzResults.png" width="50%">
10. Select any learning path from the result
11. <img src="../main/MediaFiles/Azure/AzLearningPath01.png" width="40%">
12. Start learning.
